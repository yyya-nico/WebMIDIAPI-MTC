# WebMIDIAPI-MTC
Web MIDI APIを利用して、ブラウザーからMTCをSendします。ブラウザーがMTC Senderになります。  
MTC ReceiverになるWaveTone等のために利用するイメージ。

## For Developers
```
git clone https://github.com/yyya-nico/WebMIDIAPI-MTC.git
cd ./WebMIDIAPI-MTC
npm install
```

### 開発モードで起動
```
npm run dev
```

### build
```
npm run build
```
